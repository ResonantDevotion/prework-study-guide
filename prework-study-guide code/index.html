Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML commands are in angle brackets. Most need opening and closing tags.
  • !DOCTYPE indicates to the computer what type of doc. and must be at the very start.
  • HTML is then written with a descriptor of the language.
  • HTML is all about adding structure to the webpage.

CSS

  • CSS is all about style. This makes it visually appealing to the user.

Git

  • Git is an open source platform for developer project collaboration and storage.
  • Git allows developers to work on the same project simultaneously without interfering with each others work.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • "git checkout (XXX)": checks out of the current branch to go to (XXX)
  • "git pull origin main": pulls the most updated copy of the main branch
  • "git push origin (feature/starter-code)": pushes the branch being worked on in brackets up to the main branch

JavaScript

  • Javascript adds functionality to the other features built into the webpage by HTML and CSS.

Terminal Commands

  • add -A: commmits all files in the working branch
  • -m: associates a message w our commit. This should be done to allow team members to see our update. Should be short and descriptive.
  • cd: change directory
  • .: all
  • code .: open VS Code
